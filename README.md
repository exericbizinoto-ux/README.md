# 🔐 Simulação de Ataques de Força Bruta com Kali Linux e Medusa

## 📌 Descrição do Projeto

Este projeto demonstra a realização de **ataques de força bruta em ambiente controlado** utilizando o **Kali Linux** e a ferramenta **Medusa** contra serviços vulneráveis presentes no **Metasploitable 2** e na aplicação **DVWA (Damn Vulnerable Web Application)**.

O objetivo é simular cenários de auditoria de segurança, entender como funcionam ataques de força bruta e aplicar medidas de mitigação para proteger sistemas.

⚠️ Todos os testes foram realizados **exclusivamente em laboratório para fins educacionais**.

---

# 🎯 Objetivos

- Compreender ataques de **força bruta**
- Utilizar ferramentas de **Pentest no Kali Linux**
- Realizar testes em serviços vulneráveis
- Identificar falhas de segurança
- Propor medidas de mitigação
