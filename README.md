Sobre o Projeto

Este projeto demonstra a simulação de ataques de força bruta em um ambiente de laboratório controlado, utilizando Kali Linux e a ferramenta Medusa contra serviços vulneráveis no Metasploitable 2 e na aplicação DVWA.

O objetivo é compreender como esses ataques funcionam e apresentar medidas de mitigação para aumentar a segurança dos sistemas.

⚠️ Todos os testes foram realizados apenas para fins educacionais.
